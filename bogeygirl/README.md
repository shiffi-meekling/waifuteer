# Bogeygirl
bogeygirl is a parsing library in similar flavour as boost::spirit, but for c++23. 'bogeygirl' is derived from bogeyman like a ghost or spirit, but 'girl' because I'm a girl.

by the same author as waifuteer

# installation
copy `bogeygirl.hpp` into your project folder or where your compiler will see the headers.
copy `gun.hpp` to use the gun namespace
